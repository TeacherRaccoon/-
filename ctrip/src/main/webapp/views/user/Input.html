【投影盛典-美国小镇大街】

2018年3月15日起，美国小镇大街将全新上演一场绽放米奇迷人魅力的光之盛演——“We Love Mickey!” 大街投影盛演！米妮等一众迪士尼朋友的秘密号召，送给米奇一场惊喜派对。炫目光影投射在大街两旁的房子上，将整条美国小镇大街幻化成米奇星光大街，

展现米奇演艺生涯中的星光历程。每天放映时间可能会变动，烦请查看香港迪士尼官网表演时间。

 

 

【充满活力色彩的加利布尼市集】

从色彩鲜明的布料及以珠子编成的各类织品、充满挑战的刺激游戏、美味的地道小吃，以及悦耳的音乐，全部等着宾客前来领略当中乐趣。通过玩游戏可以获得迪士尼奖品，奖品以现场供应为准。

 

【钢铁侠飞行之旅】

迪士尼乐园首个漫威主题游乐设施“铁甲奇侠飞行之旅--由友邦呈献”已于2017年1月11日登陆香港迪士尼乐园的明日世界。登上“斯塔克工业公司”最创新的科技发明-“铁甲号”飞行车，并戴上“史达前瞻智能眼镜”，与“钢铁侠1”穿梭仿似西九龙及香港闹市，飞越多个香港特色景点，例如香港史达大楼、青马大桥、维多利亚港及本地的连绵山脉等！

 

香港迪士尼乐园：

香港迪士尼乐园度假区由一个乐园和三间酒店（香港迪士尼乐园酒店、迪士尼好莱坞酒店和迪士尼探索家度假酒店）组成，致力为宾客提供难忘的奇妙迪士尼体验，让所有家庭及不同年龄的宾客乐在其中。乐园度假区于2005年9月开幕，至今已接待逾数千万来自世界各地的宾客，成为香港其中一个极受欢迎的旅游热点。

 

香港迪士尼乐园度假区持续扩建，除了美国小镇大街，探险世界，明日世界和幻想世界外，三个新的园区：反斗奇兵大本营，灰熊山谷和迷离庄园，分别于2011、2012和2013年落成；现在已有七大主题园区，提供超过一百项游乐设施及娱乐体验。香港迪士尼乐园度假区更不断推出全新的精彩演出、游乐设施及非凡互动体验,诚邀你亲临香港迪士尼乐园度假区，畅玩奇妙欢乐的迪士尼王国！

“星球大战：入侵明日世界”

传说，抵抗组织1暗中建立指挥站；随即引发第一军团2派兵展开搜索！紧急警报：激烈战幔，一触即发！香港迪士尼乐园带你投入星战前线，化身成机师、抵抗组织1英雄或绝地武士，参与这一场哄动整个银河系的惊险对决。星战极速穿梭：一场星战超时速之旅，带你穿梭于《星球大战》的激烈战火之中！ “飞越太空山”的极速旅程全新演绎，让你置身于《星球大战》一幕幕经典战争场面之中：X翼战机3与TIE战机4就在你身边互相狙击较量！

X翼战机3：星战极速穿梭的入口等候区已被打造成反抗军的机库，放置了真实大小的X翼战机3及配备机师的飞行服和头盔，为你呈献《星球大战》中的反抗世界。

 

星球大战:指挥站

抵抗组织的秘密基地，正是多位星战英雄出没的地方。大家赶紧加快脚步，与身经百战的伍基人战士丘巴卡8和灵活机智的太空技工机器人R2-D2见面。

 

第一军团巡逻侦察

法斯马上尉率领一队精锐的冲锋队，随时出现于明日世界巡逻，搜索抵抗组织的踪迹。

品尝星战美食

《星球大战》热浪延伸至味觉，乐园推出一系列新款主题美食：从全新推出的“黑武士汉堡包套餐”到“星战光剑玉桂条”，让你在舌尖也来一场甜美角力。乐园酒店餐厅呈献的“星球大战金腿五仁酥”、“星球大战三文治餐”等，能让整个银河系味觉苏醒，都是你不可错过的品尝之选。

 

太空驿站及星战精品

星战魅力，席卷乐园。太空驿站让来自整个银河系的宾客自制光剑，从此，大小宾客可以拥有属于自己的光剑，抵抗敌人。乐园推出多款独家《星球大战》Tsum Tsum 英雄系列毛公仔。同时，全新的光剑、战机、以及太空技工机器人BB-8和R2-D2玩具已炫酷登场！

 

迪士尼魔法书房‧全新百老汇式音乐剧

万众期待的全新音乐剧“迪士尼魔法书房”是特别为香港而创作、乐园历年最大型的娱乐表演制作，由米奇老鼠和高飞带领宾客穿梭7个迪士尼故事，透过华丽表演与崭新舞台技术，投入一个新层次的奇妙旅程。不论音乐、编舞、服饰及舞台设计等都融入了经典及现代元素。

 

童话园林–由潘多拉呈献‧再现经典童话场景

乐园另一别创新格的奇妙舞台，正是全新的“童话园林”– 多个迪士尼公主的经典故事场景，幻化成小模型，散布在绿树掩映、花园锦簇的园林之中，让你以别开生面的方式欣赏五位公主的童话故事。你可透过园林中的音乐盒，启动各个迪士尼公主的经典故事场景，并重温耳熟能详的优美旋律。走着走着，你会发现自己穿梭于巨型的奇花异卉之间，这里正是奇妙仙子的家 –“仙女小亭园”。奇妙仙子会于“仙女小亭园”挥洒奇妙星粉，等待与宾客会面。

 

反斗奇兵大本营

“反斗奇兵大本营”是个令人乐而忘返的乐园，宾客仿如缩小至玩具般大小，就如受欢迎的迪士尼‧皮克斯动画系列《玩具总动员》中的玩具老朋友一样。反斗奇兵大本营的游乐设施和娱乐节目带给你无限快乐，引发你无限的想像力。

 

灰熊山谷

扩建项目中最大的主题园区“灰熊山谷”以原创西部故事为设计蓝本，“灰熊山谷”小镇有七大必到西部景点，将为宾客带来一段意想不到的穿山历险。独特创新的双向过山车– “灰熊山极速矿车”依着地形山势穿过整个“灰熊山谷”，沿着多路轨纵横交错，齐来感受西部最疯狂的过山车冒险旅程！

 

迷离庄园

主题园区“迷离庄园”是关于故事主人亨利爵士与他的助手猴子阿拔游历世界各国去收集罕有珍品，收藏于其私人博物馆 －“迷离大宅”。宾客可坐上无轨的“迷离庄园电磁厢车”，探索博物馆每个角落。 “迷离大宅”于世界主题娱乐协会(TEA) 第二十届主题娱乐大奖 (Thea Awards)中，勇夺 “杰出成就奖 － 游乐设施”的殊荣。

 

“迪士尼光影汇”夜间巡游

香港迪士尼乐园是全球首个迪士尼乐园推出全LED花车夜间巡游 ─ “迪士尼光影汇”夜间巡游。光影闪烁效果除了布满主题花车上，更体现在迪士尼朋友和巡游表演者身上。超过74万颗LED灯制造出逾25种光影效果应用于服饰设计及制作上，当中不少是因应此夜间巡游而特别设计。 

 

中西荟萃的度假区

香港是个著名的中西合璧国际大都会。而在香港迪士尼乐园度假区，宾客也可欢度每个中方或西方节日，购买独特的商品，更可品尝到各国美食。每一个中方或西方节日，乐园都能给你一个奇妙难忘的佳节!

 

宾客更可在香港迪士尼乐园度假区内品尝到各国特色美食，由中餐佳肴到西式美食、香港道地特色美食，如点心、鱼蛋和蛋挞等，应有尽有，一定能满足你的味蕾。

香港迪士尼乐园度假区尊重宾客不同的国借和宗教信仰，特别为不同需要的宾客设有不同的贴心服务，尤其对穆斯林宾客的特别餐饮要求，度假区更推出设有清真认证的食品，以及娱乐设施辅助物品，将来还会根据宾客的需要提供更多贴心服务。

记得要预备足够时间，体验中西荟萃的香港迪士尼乐园度假区！

 

魔法化妆厅-让小宝贝做一回迪士尼公小公主

踏进维多利亚式公主皇室庭园内的魔法化妆厅，让仙女教母实习生施展魔法，将小朋友摇身一变成为雍容华贵的迪士尼小公主！
